MileageMonkey v0.95015 (based on JPH's 0.9 & 1.1 versions) / Tool information / Change log / Flyertalk discussion thread / Copyright © (see info)

This utility will calculate the # of miles needed for a Frequent Flyer award, and also help plan a OneWorld Explorer ticket
 
This is an updated version of the tool, loaded into place by SLF. This version was never issued by JPH. The discussion thread above should be used for feedback about this tool. Alternatively, the author may be contacted at jordan_hargrave AT hotmail.com although please bear in mind that he may have no knowledge of the modifications made in this particular version. The original site should be checked periodically to see if the tool has been reloaded there. For other versions of the tool, please see here.

  • Added valid/invalid status indicator next for OWE planner section to allow quick determination of status
  • Fixed "link to this itinerary to preserve entered "-x", "-o" values etc.
  • Streamlined the layout a little, incl. changing "Submit" to "Go!" and "Help" to "?" on buttons
  • Added more information to the help dialogs
